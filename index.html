Features Overview

Here's a list of main features and functionality included with Archcraft OS.

01

BIOS & UEFI Support

Archcraft OS works out of the box on both BIOS & UEFI systems. Users have a choice of grub or systemd-boot for UEFI installations.

02

Offline Installer

Archcraft OS uses ABIF (Arch Base Installation Framework) as the installer framework. It's a generic offline installer for Arch-based ISOs including features such as BIOS and UEFI support, multiple languages support, LUKS (LUKS, LUKS on LVM, and LVM on LUKS), LVM, btrfs (including special mounting), support for multi-user systems and built-in error detection. Archcraft OS can be installed on a hard-drive (internal or external), USB flash drive, SD card, Virtualbox, VMware, etc.

03

Custom Desktop Environment

Archcraft OS does not utilize a traditional desktop environment. Instead it installs with Openbox and Bspwm, both are window managers. Each window manager is highly configured, to give you the feel of a complete desktop environment experience. There are multiple style packages available for each Window Manager that control everything (background, panel, menus, theme , icons, fonts, etc.) with just one click. It's completely up to you to choose the style you prefer.

04

File Manager & Network Functionality

Archcraft OS ships with a pre-configured network manager and file manager. This provides users with a GUI display, access other partitions, removable media, network shares, icon thumbnailing, archive and extract compressed files, etc. It might not sound like much, but ask any new Arch user and they'll tell you this can be tricky to set up.

05

Lightweight Applications

Archcraft OS ships with a select few lightweight core applications including Thunar file manager, Midori web browser, Termite terminal interface, Geany IDE, Leafpad text editor, Viewnior image viewer, Mplayer media player. Archcraft OS is compatible and configurable with any Arch/Linux software and these pre-installed defaults can be changed.

06

Many Useful Tools

Archcraft OS includes some tools and utilities pre-installed. Shell applications like htop & bashtop (system monitors), ranger (cli fm), vim & nano (cli text editors), mpd & ncmpcpp (cli music players), zsh shell with OMZ framework & a lots of scripts and tools.
