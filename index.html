Archcraft OS

Yet another minimal Linux distribution, based on Arch Linux.

What exactly is this?

Just another Linux distribution, made on top of Arch Linux with pre-configured settings and lightweight applications.

Minimal

Archcraft uses lightweight apps, which makes it super minimal & fast. It only takes 200Mb of memory.

Stunning

User interface in Archcraft is super clean. It's minimal, beautiful and gives you multiple styles to use.

Pure

Archcraft is built on top of Arch Linux and it's pure Arch Linux. No custom kernel or anything else.


More Software

An AUR helper (yay) is already installed to help you get the latest and amazing software from Arch user repository.

Only WMs

Archcraft has no desktop environment, just some lightweight but highly configured window managers.

Free

Yeah, it's free. Free to download, free to try, free to install and absolutely free to use. Free forever.

Available Options

You can download Archcraft from either github releases, sourceforge or by torrent.

Verify the ISO


After downloading the ISO, you should verify it's authenticity.

  • Download GPG singnature and Sha256sum files. Save them in the same directory as ISO.

  • Open the terminal and Verify the details of the key on keyserver

$ gpg --recv-keys 7DC81F73
  • Verify the gpg signature by running...

$ gpg --verify archcraft-2021.06.06-x86_64.iso.sig
  • Finally, Verify the sha256sum checksum by running...

$ sha256sum -c archcraft-2021.06.06-x86_64.iso.sha256sum

Desktop Environments

Archcraft XFCE

Archcraft with XFCE Desktop Environment

Xfce is a lightweight desktop environment for UNIX-like operating systems. It aims to be fast and low on system resources, while still being visually appealing and user friendly. With Archcraft, You'll get a fully configured XFCE desktop with all of it's goodies, A clean and minimal user interface and multiple styles, Just like Archcraft openbox and bspwm.
