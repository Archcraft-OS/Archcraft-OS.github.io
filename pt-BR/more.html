Ciclo de Lançamentos

Assim como o Arch Linux, o Archcraft também é uma distribuição rolling release. Isso significa que não há necessidade de baixar uma nova ISO para atualizar o seu sistema para uma nova versão. Usuários atuais podem atualizar o sistema executando o comando sudo pacman -Syyu

Novas ISOs serão lançadas trimestralmente, em qualquer data de janeiro, abril, julho e outubro.
Lançamentos extras só ocorrerão se surgirem bugs ou problemas significativos que exijam atenção imediata.
